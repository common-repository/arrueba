Plugin que converte @name em links para o Twitter.

== Description ==

O Arrueba procura por todos os '@name' em posts e comentários e os transforma em links para o Twitter. Exemplo: "@lenonm" num post é tranformado em <a href="http://twitter.com/lenonm">@lenonm</a>.

== Installation ==

Faça upload dos arquivos do Arrueba para diretório de plugins do WordPress. Depois vá até o painel de administração e ative o plugin. Só!
